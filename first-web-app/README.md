## Features

- [x] Login & Register
- [x] Add income & expenses
- [x] View current month expenses & income
- [x] Delete expenses
- [x] Data visualization of current month expenses in a pie-chart
- [x] Compare current month expenses with another month
- [x] Data visualization of savings across months
